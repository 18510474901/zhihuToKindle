还不算入门，因为要学习的知识还很多，不只是python的基本语法。例如你要写个天气预报的小程序（暂时抛开界面库，带界面的可以参考：如何学习 Python GUI 编程？ - 用户界面设计），用于在屏幕上输出一些天气信息。1.你需要知道哪些网站提供天气API（还需要了解什么是API）。2.你需要知道怎么样通过python去获取这些接口来获取天气信息。3.你需要知道获取的这些信息需要怎么样解析（json 或者xml是什么）。如百度的：http://api.map.baidu.com/telematics/v3/weather?location=%E5%B9%BF%E5%B7%9E&output=json&ak=KPGX6sBfBZvz8NlDN5mXDNBF&callback=查询广州的天气，返回json格式，然后解析内容（上面链接点开就知道了）。例子（具体更多内容自己参考着去实现）：#!/usr/bin/env python
#coding=utf-8 
import httplib
import json

httpClient = None 
try:
    httpClient = httplib.HTTPConnection('api.map.baidu.com', 80, timeout=30)
    httpClient.request('GET', '/telematics/v3/weather?location=%E5%B9%BF%E5%B7%9E&output=json&ak=KPGX6sBfBZvz8NlDN5mXDNBF&callback=')
 
    response = httpClient.getresponse()   
    s = json.loads(response.read());
    print s["results"][0]["currentCity"]
    print s["results"][0]["weather_data"][0]["date"]
    print s["results"][0]["weather_data"][0]["weather"]
    print s["results"][0]["weather_data"][0]["wind"]
    print s["results"][0]["weather_data"][0]["temperature"]

except Exception, e:
    print e
finally:
    if httpClient:
        httpClient.close()

除了需要本身，还要学习库，也就是工具的使用。拿iOS开发做比较吧，语言层面是C/OC，它告诉你：类怎么定义、变量怎么定义、循环用for、判断用if等。它说的就是代码上你该如何去做。可是你还不知道“该做什么”，“怎么做”是发挥不出来的。所以再上一层是程序内的功能。比如天气数据，那你得拿到数据，那就要网络请求，那你得知道HTTP、xml、json。这种东西是在计算机世界里通用，约定好的，它既不属于语言范畴也不属于平台范畴（win mac ios 等）。类似的还有数据库、图形学、多媒体等，这种东西感觉是一个中间层，连结普通用户的功能需求和实际代码。你不知道怎么做，应该是对这一层了解少，没有建立起概念。这些东西都是常用的东西，所以都会被封装成一个个库来使用，所以只需要找到相应的库用起来就好了。之后再理解原理内容。
